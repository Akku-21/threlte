This component is used to consume the context provided by the `<Canvas>` component outside of it.

### Example

```svelte
<script>
  import { Canvas, ContextBridge } from '@threlte/core'

  let ctx
  $: console.log(ctx)
</script>

<Canvas>
  <ContextBridge bind:ctx />
</Canvas>
```

### Bindings

```ts
ctx: ThrelteContext
```
