# Primitive

This component can be used to insert an instance of a three.js object into the scene graph.

```svelte
<script lang="ts">
  import { Primitive } from '@threlte/core'
  import { BoxGeometry, Mesh, MeshStandardMaterial } from 'three'

  const geo = new BoxGeometry()
  const mat = new MeshStandardMaterial()
  const mesh = new Mesh(geo, mat)
</script>

<Primitive object={mesh} />
```

Just like the component `<Three>`/`<T>` it can handle any property that is available on the provided `object`.

```svelte
<Primitive
  object={mesh}
  position={[10, 5, 1]}
/>
```
